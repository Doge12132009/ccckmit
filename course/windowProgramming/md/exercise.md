## git 指令

在你 clone 下來的自己 fork 專案上，於 wp106a 資料夾下，執行

```
$ git add -A
$ git commit -m "add exercise xx-xxxxxx"
$ git push origin master
```

然後就輸入帳號密碼，上傳到你 github 自己 fork 的那個專案了。
